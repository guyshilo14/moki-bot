# 🤖 MymokiBot — הוראות הפעלה

## מה הבוט עושה?

### תזכורות קבועות (אוטומטי):
- ☀️ **7:00 בוקר** — קריאטין + מים
- 🌙 **20:00 ערב** — לקרוא מהספר

### תזכורות דינמיות:
- שעה לפני כל אירוע ⏰
- 30 דקות לפני כל אירוע ⏰

---

## איך להפעיל — Railway (הכי קל, חינם)

### צעד 1: קבל את ה-chat ID שלך
1. הפעל את הבוט (`/start`)
2. עבור לאתר: https://api.telegram.org/bot**TOKEN**/getUpdates
   (החלף TOKEN בטוקן שלך)
3. מצא את `"chat":{"id":XXXXXXX}` — זה ה-OWNER_CHAT_ID שלך

### צעד 2: העלה ל-Railway
1. לך ל: https://railway.app
2. הרשם עם GitHub (חינם)
3. לחץ **New Project** → **Deploy from GitHub repo**
4. העלה את הקבצים (bot.py, requirements.txt, Dockerfile)
5. תחת **Variables** הוסף:
   ```
   BOT_TOKEN=הטוקן שלך
   OWNER_CHAT_ID=המספר שמצאת
   ```
6. לחץ **Deploy** ✅

### צעד 3: בדוק שעובד
שלח `/start` לבוט בטלגרם — אם קיבלת תגובה, הכל עובד! 🎉

## פקודות שהבוט מבין:

| מה לכתוב | מה קורה |
|---|---|
| `תזכיר לי לדבר עם רונית ביום חמישי ב-13:00` | שומר אירוע + תזכורת |
| `עבודה אצל כהנים ביום שישי ב-8` | שומר עבודה |
| `פגישה עם דני מחר ב-10:30` | שומר פגישה |
| `מה יש לי היום?` | רשימת אירועים היום |
| `מה יש לי השבוע?` | לוח שבועי |
| `מחק 123456` | מוחק אירוע לפי ID |

## הפעלה מקומית (לבדיקה)

```bash
cd moki_bot
pip install -r requirements.txt
export BOT_TOKEN="הטוקן שלך"
export OWNER_CHAT_ID="המספר שלך"
python bot.py
```
